# 🎬 DVD Rental Profit Analysis (Australia & Canada)

## 📌 Overview
This project presents an interactive Tableau dashboard analyzing DVD rental profitability in Australia and Canada.  
It explores revenue, profit distribution, store performance, and key drivers such as film categories and actors.

---

## 🎯 Objectives
- Analyze total revenue and profit from DVD rentals  
- Compare performance between Australia and Canada  
- Evaluate profit contribution by store and category  
- Identify top-performing actors based on profitability  

---

## 🧰 Tools
- Tableau Desktop  
- Microsoft Excel   

---

## 📊 Key Insights
- Total Income: 67,407  
- Total Profit: 20,195  
- Profit is strongly driven by specific film categories and actors  
- Store performance is uneven across locations  
- Australia and Canada show different profit distribution patterns
